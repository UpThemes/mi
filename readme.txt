Mi is a responsive photoblogging theme.

== Description ==

Mi is a responsive photoblogging theme that makes displaying your photography dead simple and amazingly beautiful. Mi offers crisp typography with great readability and resizable media for any device or screen size.

Developed by [UpThemes](http://upthemes.com). 

== Installation ==

1. [Download Mi](https://dashboard.upthemes.com)
2. Install the Mi theme by going to Appearance > Themes > Add New > Upload and select the mi.zip file and click the "upload" button.
3. Activate the theme.
4. Visit the theme options page by going to Appearance > Theme Options
